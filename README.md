# Installation
###### Require python2.7

```
pip2 install -r requirements.txt
```
# Usage
```
./dhcpeyes.py -i wlan0 -t DHCPR  # Intercept only DHCPREQUEST
./dhcpeyes.py -i wlan0  # Intercept all
```
